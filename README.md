# Anymatix BYOM — Bring Your Own Magic

Share workflow collections with the Anymatix community.

Place your workflow JSON files in the `anymatix/` folder. The Anymatix app imports only those files — nothing else runs automatically.

See [PROGRAMMING_MANUAL.md](./PROGRAMMING_MANUAL.md) for a plain-language guide.

## Quick start

1. Fork this repository on GitHub.
2. Edit or add files under `anymatix/*.json`.
3. Other users discover your fork in Anymatix → Collections → Community Collections → **Follow**.

Official Anymatix collections ship inside the app. This repository is the template for **community** collections.
